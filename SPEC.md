# 鼠标点击模拟工具 - 技术规格说明书

## 1. 项目概述

### 项目名称
Simulated_Click - 窗口坐标鼠标点击模拟工具

### 项目简介
基于Python Tkinter开发的可视化鼠标点击模拟工具，支持绑定目标窗口、拾取窗口内相对坐标、模拟鼠标左右键点击，适用于自动化测试、游戏辅助等场景。

### 核心功能
- 窗口选择与绑定
- 窗口内相对坐标拾取
- 鼠标左键/右键点击模拟
- 任务列表管理与批量执行

## 2. 技术栈

| 组件 | 技术选型 | 说明 |
|------|----------|------|
| 图形界面 | Tkinter | 原生GUI，中文支持 |
| 鼠标控制 | pynput | 鼠标监听与模拟 |
| 窗口操作 | pywin32(win32gui) | 窗口获取、位置计算 |
| DPI适配 | ctypes + win32api | 高DPI感知、坐标缩放 |

## 3. 功能模块

### 3.1 窗口选择模块
- 列出当前所有可见窗口
- 支持按窗口标题搜索筛选
- 绑定选定窗口并实时跟踪位置
- 窗口丢失检测与提示

### 3.2 坐标拾取模块
- 全局热键触发拾取（默认：Ctrl+F1）
- 获取鼠标在目标窗口内的相对坐标
- 实时显示拾取坐标预览
- 支持手动输入坐标

### 3.3 任务管理模块
- 添加点击任务（倒计时、X、Y、点击类型）
- 编辑已有任务
- 删除选中任务
- 任务列表排序与展示

### 3.4 执行引擎模块
- 按顺序执行任务列表
- 实时倒计时显示
- 窗口位置动态跟踪
- 执行进度可视化

## 4. 数据结构

### 任务对象 (ClickTask)
```python
{
    "id": int,              # 任务序号
    "countdown": float,     # 倒计时（秒）
    "relative_x": int,      # 窗口内相对X坐标
    "relative_y": int,      # 窗口内相对Y坐标
    "click_type": str       # "左键点击" / "右键点击"
}
```

### 窗口对象 (TargetWindow)
```python
{
    "hwnd": int,            # 窗口句柄
    "title": str,           # 窗口标题
    "rect": tuple,          # 窗口矩形 (left, top, right, bottom)
}
```

## 6. 异常处理

| 异常场景 | 处理方式 |
|----------|----------|
| 倒计时为负数或非数字 | 弹窗提示"倒计时必须为正数" |
| 坐标为负数 | 弹窗提示"坐标必须为非负整数" |
| 目标窗口不存在 | 弹窗提示"目标窗口已关闭"，终止执行 |
| 窗口句柄无效 | 弹窗提示"窗口失效，请重新选择" |
| 重复点击开始 | 忽略本次点击，状态提示"执行中..." |
| pynput初始化失败 | 弹窗提示并退出程序 |

## 7. DPI适配方案

### 高DPI感知设置
```python
ctypes.windll.shcore.SetProcessDpiAwareness(2)  # PROCESS_PER_MONITOR_DPI_AWARE
```

### 坐标转换
- 使用 win32gui.GetWindowRect 获取窗口位置
- 使用 win32gui.ScreenToClient 转换坐标
- 使用 pynput.mouse.Controller.position 获取/设置全局坐标
- 所有坐标乘以缩放因子校正
