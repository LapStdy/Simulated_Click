# 鼠标点击模拟工具 v1.0

## 项目介绍
一个基于Python Tkinter开发的可视化鼠标点击模拟工具，支持绑定目标窗口、拾取窗口内相对坐标、模拟鼠标左右键点击，可用于自动化测试、游戏辅助等场景。

## 功能特点
- **窗口选择与绑定** - 选择并绑定目标程序窗口
- **坐标拾取** - 鼠标移到目标窗口内，按Ctrl+F1快捷键拾取相对坐标
- **任务管理** - 支持添加、编辑、删除点击任务
- **序号刷新** - 点击「刷新序号」重排任务序号
- **自动换算** - 执行时实时读取窗口位置，窗口移动不影响点击精度
- **DPI适配** - 支持Windows高DPI缩放，坐标精确无偏移
- **配置保存** - 自动保存窗口和任务配置，下次打开自动恢复
- **最小化窗口** - 执行任务前最小化窗口，防止遮挡目标程序
- **控制选项** - 自动关闭、自动执行、最小化窗口等选项
- **日志显示** - 实时显示操作日志和执行进度

## 快速开始
### 方式一：直接运行EXE
1. 下载 `鼠标点击模拟工具_v1.0.exe`
2. 双击运行（无需安装Python）

### 方式二：运行源代码
1. 安装依赖：
```bash
pip install -r requirements.txt
```

2. 运行程序：
```bash
python simulated_click.py
```

### 使用步骤
1. 点击「刷新」获取当前窗口列表
2. 选择目标窗口后点击「选择目标窗口」绑定
3. 点击「拾取窗口坐标」或「添加动作」添加点击任务
4. 调整倒计时、坐标和点击类型
5. 点击「开始执行」运行任务

## 系统要求
- **操作系统**：Windows 7/8/10/11
- **Python版本**：Python 3.7+（仅运行源代码时需要）
- **其他依赖**：pynput、pywin32

## 相关说明
### 配置文件
程序首次运行后会在同一目录下生成 `config.json` 配置文件，保存当前的窗口、任务和设置。

### 快捷键
- **Ctrl+F1**：拾取鼠标在目标窗口内的相对坐标

### 注意事项
- 点击操作可能触发目标程序的安全机制，请在合适的场景使用
- 窗口标题变化可能导致绑定失效，需要重新选择
- 高DPI缩放下，窗口内相对坐标可能会变化，建议重新拾取

## 常见问题
Q: 快捷键Ctrl+F1没有反应？
A: 请确保目标窗口已绑定，并且「拾取窗口坐标」对话框处于打开状态。

Q: 窗口移动后点击位置不对？
A: 程序会自动重新计算坐标，请确保任务执行时目标窗口保持可见。

Q: 配置没有保存？
A: 请确保程序正常关闭，或者点击「保存设置」按钮手动保存。

Q: EXE运行时杀毒软件报毒？
A: 这是因为程序模拟鼠标操作，属于正常现象，请添加信任。
